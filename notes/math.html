Ellipse

(x2)/(a2) + (y2)/(b2) = 1, where a > b and the focus on x axis.

x = acosθ and y = bsinθ.

The focus are ±c = ±a×e, where the eccentricity is e = √(1 − (b2)/(a2)).

Parametric ellipse

Or Trammel of Archimedes is the definition of a ≡ p + q and b = q.

If q ≡ 1, then p = √(1 − e2) − 1.

Rotation matrix (Euler angles α, β, γ)

Rx = ⎡⎢⎢⎢⎣ 1 0 0       0 cosα  − sinα       0 sinα cosα ⎤⎥⎥⎥⎦
Ry = ⎡⎢⎢⎢⎣ cosβ 0 sinβ       0 1 0        − sinβ 0 cosβ ⎤⎥⎥⎥⎦
Rz = ⎡⎢⎢⎢⎣ cosγ  − sinγ 0       sinγ cosγ 0       0 0 1 ⎤⎥⎥⎥⎦
R = Rx⋅Ry⋅Rz

In python:

R = np.dot(Rz,(np.dot(R_x,R_y)))
